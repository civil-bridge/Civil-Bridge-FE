# 마이룸(내 정보) 화면 설계서

**화면 설명**: 사용자의 개인 정보를 관리하고 공무원 인증을 진행할 수 있는 페이지입니다.

**진입 경로**: 헤더 → '마이룸' 링크 클릭

---

## 화면 요소

| 요소 | 타입 | 설명 | 구현 상세 |
| --- | --- | --- | --- |
| 뒤로가기 버튼 | Button | 메인 화면으로 이동 | `ChevronLeft` 아이콘 사용 |
| 프로필 섹션 | Section | 프로필 이미지 및 기본 정보 | 원형 이미지(또는 이름 첫 자), 닉네임, 이메일 표시 |
| 회원정보 수정 섹션 | Form | 닉네임, 이메일, 비밀번호 수정 | 각 필드별 입력창 및 `변경사항 저장` 버튼 |
| 공무원 인증 섹션 | Section | 공무원 신분 인증 절차 | 인증 메일 입력창, `인증 메일 발송` 버튼 |
| 공무원 인증 상태 | Badge | 인증 여부 표시 | 인증 완료 시 '공무원 인증 완료' 뱃지 표시 |
| 로그아웃 버튼 | Button | 세션 종료 및 로그인 화면 이동 | 빨간색 텍스트 스타일, 확인 모달 포함 |

---

## 사용자 동작 및 인터랙션

| 동작 | 결과 | 예외 처리 및 피드백 |
| --- | --- | --- |
| 변경사항 저장 클릭 | 정보 업데이트 모의 실행 | "변경사항이 저장되었습니다" 얼럿 표시 |
| 인증 메일 발송 클릭 | 인증 메일 발송 안내 모달 오픈 | 메일 형식 확인 및 발송 성공 안내 |
| 로그아웃 클릭 | 로그아웃 확인 모달 오픈 | '확인' 선택 시 `/login`으로 이동 |
| 뒤로가기 클릭 | 메인 화면('/')으로 이동 | - |

---

## 디자인 시스템 적용
- **중앙 정렬 레이아웃**: `max-w-[640px]`의 좁은 레이아웃으로 집중도 향상
- **카드 스타일**: 섹션별 화이트 카드 배경 및 둥근 모서리(`rounded-2xl`)
- **아이콘**: `lucide-react` (Camera, CheckCircle2, Mail, LogOut) 활용
