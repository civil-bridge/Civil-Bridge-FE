# Global Common Schema

이 문서는 모든 도메인(BE/FE)에서 공통으로 보장해야 하는 데이터 규격 및 Enum 정의를 관리합니다. 
이곳의 정의가 변경되면 모든 관련 도메인의 명세와 코드가 업데이트되어야 합니다.

---

## 1. 전역 Enum 정의

### Region (행정 구역 - 2단계)
- **City (시/도)**: `서울특별시`, `부산광역시`, `대구광역시`, `인천광역시`, `광주광역시`, `대전광역시`, `울산광역시`, `세종특별자치시`, `경기도`, `강원특별자치도`, `충청북도`, `충청남도`, `전북특별자치도`, `전라남도`, `경상북도`, `경상남도`, `제주특별자치도`
- **District (시/군/구)**: 각 시/도에 종속된 기초자치단체 단위 (예: 수원시, 강남구 등). 
- **제약**: 하위 행정구(예: 수원시 영통구)는 관리하지 않고 '시' 단위까지만 관리함을 원칙으로 함.

### AccessLevel (접근 권한)
| 값 | 설명 |
|----|------|
| `PUBLIC` | 모든 사용자 접근 가능 |
| `OFFICIALS_ONLY` | 인증된 공무원만 접근 가능 |
| `USER_ONLY` | 일반 시민 사용자만 접근 가능 |

### UserRole (사용자 역할)
| 값 | 설명 |
|----|------|
| `USER` | 일반 시민 |
| `OFFICIAL` | 인증된 공무원 |
| `LEADER` | 논의방 방장 (논의방 도메인 내 특수 역할) |

### ProposalStatus (제안서 상태)
| 값 | 설명 |
|----|------|
| `DRAFT` | 제안서 작성 중 (방장만 편집 가능) |
| `VOTING` | 동의 투표 진행 중 (편집 불가) |
| `COMPLETED` | 투표 가결 및 최종 제출 완료 |
| `REJECTED` | 투표 부결 또는 기한 만료 |

## 2. 공통 통신 포맷

### ApiResponse<T> (API 공통 응답)
모든 API 응답은 이 구조를 따릅니다.
```json
{
  "code": "String (SUCCESS | ERROR | INVALID_AUTH 등)",
  "message": "String (사용자 노출용 메시지)",
  "data": "T (실제 데이터 객체)"
}
```

### PageInfo (페이징 정보)
```json
{
  "currentPage": "Number",
  "pageSize": "Number",
  "totalCount": "Number",
  "totalPages": "Number"
}
```
