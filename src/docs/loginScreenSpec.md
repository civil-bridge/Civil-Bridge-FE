# 로그인 화면 설계서

## 1. 페이지 개요

- **URL**: `/login`
- **목적**: 기존 사용자 로그인

---

## 2. 페이지 구성

### 2.1. 레이아웃

```
┌─────────────────────────────────────────────────┐
│                                                 │
│              ┌─────────────────┐                │
│              │                 │                │
│              │    로그인 폼     │                │
│              │                 │                │
│              └─────────────────┘                │
│                                                 │
│          계정이 없으신가요? 회원가입               │
│                                                 │
└─────────────────────────────────────────────────┘
```

- 중앙 정렬 레이아웃
- 최대 너비: 400px
- 배경: linear-gradient(135deg, #FAF5FF 0%, #F5F5F4 50%, #FFF7ED 100%)
- 폼 컨테이너: 카드 스타일 (white, border, shadow)

### 2.2. UI 요소

| UI 요소 | 설명 |
|---------|------|
| 페이지 타이틀 | "로그인" |
| 아이디 입력 | 로그인 아이디 |
| 비밀번호 입력 | 비밀번호 (마스킹 처리, 눈 아이콘으로 보기/숨기기) |
| 로그인 버튼 | 폼 제출 |
| 회원가입 링크 | 회원가입 페이지로 이동 |

---

## 3. 입력 필드 상세

| 필드 | 타입 | 필수 | 플레이스홀더 | 유효성 검사 |
|------|------|------|--------------|-------------|
| 아이디 | text | ✓ | 아이디를 입력하세요 | 빈 값 체크 |
| 비밀번호 | password | ✓ | 비밀번호를 입력하세요 | 빈 값 체크 |

---

## 4. 사용자 상호작용

| 상호작용 | 결과 |
|----------|------|
| 아이디/비밀번호 입력 | 입력값 상태 업데이트 |
| 비밀번호 눈 아이콘 클릭 | 비밀번호 보기/숨기기 토글 |
| 로그인 버튼 클릭 | 유효성 검사 → 통과 시 API 호출 |
| 로그인 성공 | 메인페이지로 이동 (`/`) |
| 로그인 실패 | 에러 메시지 표시 |
| 회원가입 링크 클릭 | 회원가입 페이지로 이동 (`/signup`) |

---

## 5. 에러 메시지

| 상황 | 에러 메시지 |
|------|-------------|
| 아이디 미입력 | 아이디를 입력해주세요 |
| 비밀번호 미입력 | 비밀번호를 입력해주세요 |
| 로그인 실패 | 아이디 또는 비밀번호가 일치하지 않습니다 |
