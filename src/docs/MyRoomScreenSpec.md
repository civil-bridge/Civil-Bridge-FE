# 마이룸 화면 설계서

## 1. 페이지 개요

- **URL**: `/myroom`
- **목적**: 회원정보 수정, 공무원 인증, 로그아웃
- **접근 조건**: 로그인 필수

---

## 2. 페이지 구성

### 2.1. 헤더

```
┌─────────────────────────────────────────────────────────────┐
│  [←] 마이룸                                                  │
└─────────────────────────────────────────────────────────────┘
```

| UI 요소 | 설명 |
|---------|------|
| 뒤로가기 버튼 [←] | 메인페이지로 이동 |
| 페이지 제목 | "마이룸" |

### 2.2. 프로필 섹션

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│                    [프로필 이미지]                           │
│                      (클릭하여 변경)                         │
│                                                             │
│                       김시민                                 │
│                   example@email.com                         │
│                                                             │
│              [공무원 인증 완료] 또는 [공무원 인증하기]         │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

| UI 요소 | 설명 |
|---------|------|
| 프로필 이미지 | 원형, 클릭 시 이미지 변경 가능 |
| 닉네임 | 현재 닉네임 표시 |
| 이메일 | 현재 이메일 표시 |
| 공무원 인증 뱃지/버튼 | 인증 완료 시 뱃지, 미인증 시 버튼 |

### 2.3. 회원정보 수정 섹션

```
┌─────────────────────────────────────────────────────────────┐
│  회원정보 수정                                               │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  닉네임                                                      │
│  ┌─────────────────────────────────────┐                    │
│  │ 김시민                               │                    │
│  └─────────────────────────────────────┘                    │
│                                                             │
│  이메일                                                      │
│  ┌─────────────────────────────────────┐                    │
│  │ example@email.com                   │                    │
│  └─────────────────────────────────────┘                    │
│                                                             │
│  비밀번호 변경                                               │
│  ┌─────────────────────────────────────┐                    │
│  │ 현재 비밀번호                        │                    │
│  └─────────────────────────────────────┘                    │
│  ┌─────────────────────────────────────┐                    │
│  │ 새 비밀번호                          │                    │
│  └─────────────────────────────────────┘                    │
│  ┌─────────────────────────────────────┐                    │
│  │ 새 비밀번호 확인                     │                    │
│  └─────────────────────────────────────┘                    │
│                                                             │
│                              [변경사항 저장]                 │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

| UI 요소 | 설명 |
|---------|------|
| 닉네임 입력 | 현재 닉네임, 수정 가능 |
| 이메일 입력 | 현재 이메일, 수정 가능 |
| 현재 비밀번호 | 비밀번호 변경 시 필수 입력 |
| 새 비밀번호 | 변경할 비밀번호 |
| 새 비밀번호 확인 | 새 비밀번호 재입력 |
| 변경사항 저장 버튼 | 수정 내용 저장 |

### 2.4. 공무원 인증 섹션

```
┌─────────────────────────────────────────────────────────────┐
│  공무원 인증                                                 │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  공무원 전용 이메일로 인증하여 공무원 뱃지를 받으세요.         │
│                                                             │
│  공무원 이메일                                               │
│  ┌─────────────────────────────────────┐                    │
│  │ @go.kr / @korea.kr 등               │                    │
│  └─────────────────────────────────────┘                    │
│                                                             │
│                           [인증 메일 발송]                   │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

| UI 요소 | 설명 |
|---------|------|
| 안내 문구 | 공무원 인증 설명 |
| 공무원 이메일 입력 | @go.kr, @korea.kr 등 공무원 이메일 |
| 인증 메일 발송 버튼 | 인증 메일 발송 |

**인증 완료 상태일 때:**
```
┌─────────────────────────────────────────────────────────────┐
│  공무원 인증                                                 │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  ✓ 공무원 인증이 완료되었습니다.                             │
│    인증 이메일: official@go.kr                               │
│    인증일: 2025.01.02                                        │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 2.5. 로그아웃 섹션

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│                        [로그아웃]                            │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

| UI 요소 | 설명 |
|---------|------|
| 로그아웃 버튼 | 클릭 시 로그아웃 확인 모달 |

---

## 3. 사용자 상호작용

| 상호작용 | 결과 |
|----------|------|
| 뒤로가기 버튼 클릭 | 메인페이지로 이동 |
| 프로필 이미지 클릭 | 이미지 선택 창 열림 (파일 업로드) |
| 닉네임/이메일 수정 후 저장 | 변경사항 저장, 성공 토스트 표시 |
| 비밀번호 변경 후 저장 | 비밀번호 변경, 성공 토스트 표시 |
| 공무원 인증하기 버튼 클릭 | 공무원 인증 섹션으로 스크롤 |
| 인증 메일 발송 클릭 | 메일 발송, 안내 모달 표시 |
| 로그아웃 클릭 | 로그아웃 확인 모달 → 확인 시 로그인 페이지로 이동 |

---

## 4. 모달 목록

| 모달 | 트리거 | 내용 | 버튼 |
|------|--------|------|------|
| 로그아웃 확인 | "로그아웃" 클릭 | "정말 로그아웃 하시겠습니까?" | [확인] [취소] |
| 인증 메일 발송 완료 | "인증 메일 발송" 클릭 | "인증 메일이 발송되었습니다. 메일함을 확인해주세요." | [확인] |
| 저장 완료 | "변경사항 저장" 클릭 | 토스트 메시지: "변경사항이 저장되었습니다." | 자동 닫힘 (3초) |

## 5. 상태별 화면

### 5.1. 공무원 인증 상태

| 상태 | 화면 |
|------|------|
| 미인증 | 프로필에 [공무원 인증하기] 버튼, 인증 섹션에 이메일 입력 폼 |
| 인증 완료 | 프로필에 [공무원 인증 완료] 뱃지, 인증 섹션에 완료 정보 표시 |
