# 채팅방(토의실) 화면 설계서

## 1. 페이지 개요

- **URL**: `/room/{roomId}`
- **목적**: 실시간 채팅으로 논의하고, 방장이 제안서를 작성하는 공간
- **레이아웃**: 채팅 영역 (65%) + 제안서 패널 (35%, 기본 접힘)

---

## 2. 권한 구조

| 역할 | 채팅 | 제안서 보기 | 제안서 편집 | 참여자 강퇴 | 방 삭제 |
|------|------|-------------|-------------|-------------|---------|
| 방장 | ✓ | ✓ | ✓ | ✓ | ✓ |
| 참여자 | ✓ | ✓ (읽기 전용) | ✗ | ✗ | ✗ |

---

## 3. 페이지 구성

### 3.1. 헤더

```
┌─────────────────────────────────────────────────────────────┐
│  [←] 부천역 소음문제 해결 · 부천시 · 12명 참여중       [☰]  │
└─────────────────────────────────────────────────────────────┘
```

| UI 요소 | 설명 |
|---------|------|
| 뒤로가기 버튼 [←] | 메인페이지로 이동 |
| 토의실 제목 | 현재 토의실 이름 |
| 지역 태그 | 해당 지역 (구분자 · 로 연결) |
| 참여자 수 | "N명 참여중" |
| 메뉴 버튼 [☰] | 슬라이드 메뉴 열기 |

### 3.2. 슬라이드 메뉴

메뉴 버튼 클릭 시 우측에서 슬라이드로 열림.

| UI 요소 | 모든 사용자 | 방장만 클릭 가능 |
|---------|-------------|------------------|
| 참여자 목록 (N명) | ✓ 노출 | - |
| 제안서 열기 | ✓ 노출 | ✓ (편집 모드) |
| 방 나가기 | ✓ 노출 | - |
| 참여자 강퇴 | ✓ 노출 | ✓ |
| 방 삭제 | ✓ 노출 | ✓ |

**참여자가 "제안서 열기" 클릭 시:**
- 모달 표시: "방장만 제안서를 편집할 수 있습니다. 읽기 전용으로 보시겠습니까?"
- [예] → 제안서 패널 읽기 모드로 열림
- [아니오] → 모달 닫힘

**참여자가 "참여자 강퇴" 또는 "방 삭제" 클릭 시:**
- 모달 표시: "방장만 사용할 수 있는 기능입니다."
- [확인] → 모달 닫힘

### 3.3. 채팅 영역

| UI 요소 | 설명 |
|---------|------|
| 채팅 메시지 목록 | 스크롤 가능한 메시지 영역 |
| 메시지 입력창 | 하단 고정 |
| 전송 버튼 | 메시지 전송 |

#### 메시지 구성 요소

| UI 요소 | 설명 |
|---------|------|
| 프로필 아이콘 | 사용자 아이콘 (방장은 차별화된 아이콘/테두리) |
| 작성자 이름 | 닉네임 |
| 방장 뱃지 | 방장인 경우 "방장" 뱃지 표시 |
| 공무원 뱃지 | 공무원인 경우 "OFFICIAL" 뱃지 표시 |
| 메시지 내용 | 본문 텍스트 |
| 작성 시간 | "오후 2:00" 형태 |

### 3.4. 제안서 패널

기본 상태: **접힘**

#### 접힌 상태

제안서 패널이 화면에 보이지 않음. 슬라이드 메뉴의 "제안서 열기"로만 접근.

#### 펼친 상태 (방장 - 편집 모드)

| UI 요소 | 설명 |
|---------|------|
| 패널 헤더 | "제안서" + 닫기 버튼 [X] |
| 제목 입력 | 제안서 제목 (필수) |
| 현재 문제 입력 | 텍스트에리어 (필수) |
| 제안 솔루션 입력 | 텍스트에리어 (필수) |
| 첨부파일 영역 | 이미지/문서 첨부 |
| 임시저장 버튼 | 작성 중 저장 |
| 최종 제출 버튼 | 제안서 제출 |

#### 펼친 상태 (참여자 - 읽기 모드)

| UI 요소 | 설명 |
|---------|------|
| 패널 헤더 | "제안서" + 닫기 버튼 [X] |
| 제안서 내용 | 방장이 작성한 내용 (읽기 전용) |
| 작성 상태 | "작성중" 또는 "제출 완료" 표시 |

## 4. 사용자 상호작용

### 4.1. 공통 (모든 사용자)

| 상호작용 | 결과 |
|----------|------|
| 뒤로가기 버튼 클릭 | 메인페이지로 이동 |
| 메뉴 버튼 클릭 | 슬라이드 메뉴 열림 |
| 슬라이드 메뉴 외부 클릭 | 메뉴 닫힘 |
| 메시지 입력 후 전송 | 채팅에 메시지 추가 |
| Enter 키 | 메시지 전송 |
| Shift + Enter | 줄바꿈 |
| 채팅 영역 스크롤 | 이전 메시지 조회 |
| 방 나가기 클릭 | 확인 모달 → 확인 시 메인으로 이동 |

### 4.2. 방장 전용

| 상호작용 | 결과 |
|----------|------|
| 제안서 열기 클릭 | 제안서 패널 편집 모드로 열림 |
| 제안서 내용 편집 | 실시간 저장 또는 임시저장 |
| 임시저장 클릭 | 현재 상태 저장 |
| 최종 제출 클릭 | 확인 모달 → 확인 시 제출 |
| 참여자 강퇴 클릭 | 참여자 목록에서 선택 → 확인 모달 → 강퇴 |
| 방 삭제 클릭 | 확인 모달 → 확인 시 방 삭제, 메인으로 이동 |

### 4.3. 참여자 전용

| 상호작용 | 결과 |
|----------|------|
| 제안서 열기 클릭 | 읽기 전용 안내 모달 → [예] 시 읽기 모드로 열림 |
| 참여자 강퇴 클릭 | "방장만 사용 가능" 모달 |
| 방 삭제 클릭 | "방장만 사용 가능" 모달 |

## 7. 상태별 화면

### 7.1. 제안서 상태

| 상태 | 방장 화면 | 참여자 화면 |
|------|-----------|-------------|
| 미작성 | 빈 입력 폼 | "아직 작성된 제안서가 없습니다" |
| 작성중 | 편집 가능한 폼 + 임시저장 내용 | 작성중인 내용 읽기 전용 |
| 제출완료 | 수정 불가, 제출된 내용 표시 | 제출된 내용 읽기 전용 |
