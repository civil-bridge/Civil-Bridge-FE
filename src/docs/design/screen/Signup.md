# 회원가입 화면 설계서

**화면 설명**: 새로운 사용자가 계정을 생성하기 위한 화면입니다. 아이디 중복 확인, 이메일 인증, 닉네임 중복 확인 절차를 포함합니다.

**진입 경로**: 로그인 화면 → 회원가입 링크 클릭

---

## 화면 요소

| 요소 | 타입 | 설명 | 구현 상세 |
| --- | --- | --- | --- |
| 로고 | Image/SVG | 서비스 로고 표시 | `Logo` 공통 컴포넌트 사용 |
| 아이디 입력창 | Input | 사용할 아이디 입력 | 중복확인 버튼 포함, 최소 6자 이상 |
| 중복확인 버튼 | Button | 아이디 중복 여부 체크 | `secondary` 스타일, 확인 완료 시 '✓ 사용가능'으로 변경 |
| 비밀번호 입력창 | Input | 비밀번호 설정 | 대/소문자, 특수문자 포함 12자 이상 검증, 가시성 토글 기능 |
| 비밀번호 재입력 | Input | 비밀번호 확인 입력 | 이전 비밀번호와 일치 여부 검증 |
| 이메일 입력창 | Input | 사용자 이메일 입력 | 인증번호 발송 버튼 포함, 형식 검증 |
| 인증번호 발송 버튼 | Button | 이메일로 인증번호 전송 | 클릭 시 타이머(3분) 가동, '재발송' 기능 지원 |
| 인증번호 입력창 | Input | 전송된 6자리 코드 입력 | 인증번호 발송 후에만 노출됨 |
| 인증 확인 버튼 | Button | 코드 유효성 검증 | 인증 성공 시 '✓ 인증완료' 상태로 변경 |
| 닉네임 입력창 | Input | 활동 닉네임 설정 | 중복확인 버튼 포함, 최소 2자 이상 |
| 회원가입 버튼 | Button | 최종 가입 요청 | 모든 검증 완료 시 활성화 유도 |

---

## 사용자 동작 및 인터랙션

| 동작 | 결과 | 예외 처리 및 피드백 |
| --- | --- | --- |
| 중복확인 클릭 | 아이디/닉네임 사용 가능 여부 확인 | 유효성 미달 시 에러 메시지, 성공 시 성공 메시지 및 상태 업데이트 |
| 인증번호 발송 클릭 | 인증 이메일 발송 및 입력창 노출 | 이메일 형식 오류 시 경고 표시 |
| 인증 확인 클릭 | 입력한 코드가 '123456'인지 확인 | 만료 시 또는 불일치 시 에러 메시지 표시 |
| 회원가입 버튼 클릭 | 로그인 화면('/login')으로 이동 | 필수 확인 사항(중복확인, 인증 등) 누락 시 차단 |

---

## 디자인 시스템 적용
- 로그인 화면과 동일한 배경 및 카드 스타일 유지
- **타이머**: 인증번호 입력창 하단에 남은 시간 표시 (`formattedTime`)
- **버튼 상태**: 유효성 검사 결과에 따라 `ghost`, `secondary`, `primary` 스타일 동적 전환
