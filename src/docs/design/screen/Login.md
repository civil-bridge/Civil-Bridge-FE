# 로그인 화면 설계서

**화면 설명**: 사용자가 자신의 계정으로 시스템에 접속하기 위한 화면입니다. 깔끔한 그라데이션 배경과 중앙 정렬된 카드 레이아웃을 사용합니다.

**진입 경로**: 앱 첫 진입 시 또는 로그아웃 시

---

## 화면 요소

| 요소 | 타입 | 설명 | 구현 상세 |
| --- | --- | --- | --- |
| 로고 | Image/SVG | 서비스 로고 표시 | `Logo` 공통 컴포넌트 사용 |
| 아이디 입력창 | Input | 사용자 아이디 입력 | `Input` 컴포넌트, `placeholder="아이디를 입력하세요"` |
| 비밀번호 입력창 | Input | 사용자 비밀번호 입력 | `type="password"`, 눈 아이콘 버튼으로 가시성 토글 기능 포함 |
| 로그인 버튼 | Button | 로그인 처리 실행 | `Button` 컴포넌트, `fullWidth`, `primary` 스타일 |
| 회원가입 링크 | Link | 회원가입 화면으로 이동 | "계정이 없으신가요? 회원가입" 텍스트와 함께 배치 |

---

## 사용자 동작 및 인터랙션

| 동작 | 결과 | 예외 처리 및 피드백 |
| --- | --- | --- |
| 로그인 버튼 클릭 | 메인 화면('/')으로 이동 | 아이디/비밀번호 누락 시 입력창 하단에 에러 메시지 표시 |
| 눈 아이콘 클릭 | 비밀번호 마스킹 해제/설정 | 클릭 시마다 `password` <-> `text` 타입 전환 |
| 회원가입 링크 클릭 | 회원가입 화면('/signup')으로 이동 | - |

---

## 디자인 시스템 적용
- **배경**: `bg-gradient-to-br from-[#FAF5FF] via-[#F5F5F4] to-[#FFF7ED]`
- **카드**: `bg-white`, `p-8`, `rounded-2xl`, `border-neutral-200`, `shadow-[0_4px_12px_rgba(0,0,0,0.08)]`
- **입력창**: `Input` 공통 컴포넌트의 표준 스타일 적용
